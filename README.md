# VUT Campus Compass 🏫

Campus safety + navigation app for Vaal University of Technology students.

### **Features**
- **Navigate to VUT** - One-tap directions to main campus
- **Emergency Assembly Points** - Shows all VUT safety zones with GPS navigation  
- **Panic Button** - Instantly share location with emergency contacts via WhatsApp
- **Campus Layout** - View official VUT emergency map in-app
- **Share My Location** - Send live GPS to friends/security

### **Tech Stack**
Built with Expo + React Native
- `react-native-maps` for GPS
- `expo-location` for permissions
- EAS Build for APK generation

### **VUT Colors**
- Maroon: `#6C0A0A`
- Gold: `#FFC72C`

### **Install APK**
1. Download the latest `.apk` from Expo Builds
2. Open on Android phone → Tap Install
3. Allow "Install unknown apps" if prompted
